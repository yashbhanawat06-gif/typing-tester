🚀 Typing Speed Test

A simple web app to test typing speed, accuracy, and time taken.

🛠️ Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Render (Deployment)

⚡ Features
- Timer starts on first key press
- Calculates WPM
- Measures accuracy
- Random sentences
