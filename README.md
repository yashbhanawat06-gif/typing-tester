# 🚀 Typing Speed Test

A simple web application to test typing speed, accuracy, and time taken.

---

## 🚀 Features
- Timer starts on first key press  
- Calculates words per minute (WPM)  
- Measures typing accuracy  
- Displays random sentences  

---

## 🛠️ Tech Stack
- Python (Flask)  
- HTML5  
- CSS3  
- JavaScript
